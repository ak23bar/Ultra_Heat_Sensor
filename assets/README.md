## Image Guidelines

- **Format**: PNG for diagrams, JPG for photos
- **Resolution**: Minimum 1920×1080 for photos
- **Naming**: Use descriptive names (e.g., `ultrasonic_sensor_wiring.jpg`)
- **Size**: Optimize images (<500KB per image)

## Video Guidelines

- **Format**: MP4 (H.264 codec)
- **Resolution**: 1080p preferred, 720p minimum
- **Length**: 2-5 minutes for demos
- **Audio**: Clear narration or captions
- **Hosting**: Upload to YouTube/Vimeo and link in README

## Usage

Reference assets in documentation:
```markdown
![System Architecture](assets/images/system_architecture.png)
```

Or in HTML:
```html
<img src="assets/images/hardware_setup.jpg" alt="Hardware Setup" width="600">
```
